MXD Importer-Lite

Nhập CSV / thêm nhanh / dán hàng loạt / kéo-thả ảnh → gộp affiliates.json + ZIP ảnh .webp đúng chuẩn /assets/img/products/<sku>.webp.

1) Tải CSV

CSV Shopee/Excel có thể gồm: Mã sản phẩm, Tên sản phẩm, Giá, Link sản phẩm…

2) Tuỳ chọn


Ảnh luôn theo chuẩn: /assets/img/products/<sku>.webp

3) Hành động dữ liệu

Log



      
Thiết lập map cột

      
Có thể sửa tên cột đích: name, price_vnd, origin_url, merchant, sku
